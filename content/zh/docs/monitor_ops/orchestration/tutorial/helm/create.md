该功能用于对接Helm仓库。

{{% alert title="说明" %}}
推荐使用的Helm仓库的URL：

- 虚拟机Helm仓库：https://cloudpods-charts.oss-cn-hongkong.aliyuncs.com/
- 容器Helm仓库：https://apphub.aliyuncs.com
{{% /alert %}}


1. 在Helm仓库页面，单击列表上方 **_"新建"_** 按钮，弹出新建Helm仓库对话框。
2. 设置以下参数：
   - 名称：设置Helm仓库的名称。
   - 指定域：设置Helm仓库所属的域。
   - URL：设置Helm仓库的URL路径。
   - 类型：设置Helm仓库适用的资源类型，包括虚拟机类型和容器类型。
3. 单击 **_"确定"_** 按钮，完成操作。
