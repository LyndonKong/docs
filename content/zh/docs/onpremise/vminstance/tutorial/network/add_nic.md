description: >
    为虚拟机添加多块网卡
---

该功能用于为虚拟机添加网卡，最多可添加8个网卡，目前仅{{<oem_name>}}平台的虚拟机支持添加网卡功能。

1. 在左侧导航栏，选择 **_"主机/主机/虚拟机"_** 菜单项，进入虚拟机页面。
2. 单击虚拟机名称项，进入虚拟机详情页面。
2. 单击“网络”页签，进入网络页面。
3. 单击网卡列表上方 **_"添加网卡"_** 按钮，弹出添加网卡对话框。
4. 设置VPC和IP子网，如需指定静态IP，可单击 **_"手动配置IP"_** 按钮，设置IP地址，单击 **_"确定"_** 按钮。
