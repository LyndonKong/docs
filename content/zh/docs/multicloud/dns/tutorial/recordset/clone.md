该功能用于基于当前解析记录创建新的记录。

1. 在左侧导航栏，选择 **_"网络/网络服务/DNS解析"_** 菜单项，进入DNS解析页面。
2. 单击域名名称项，进入DNS解析详情页面。
2. 单击“记录”页签，进入记录页面。
3. 单击记录右侧操作列 **_"更多"_** 按钮，选择下拉菜单 **_"克隆"_** 菜单项，弹出克隆对话框。
4. 配置相关信息，单击 **_"确定"_** 按钮，创建新的域名。
