该功能用于将虚拟机与密钥解除绑定。

{{% alert title="注意" color="warning" %}}
- OpenStack平台虚拟机解绑密钥后重置密码可能会报错，请手动重置密码。
- Nutanix不支持解绑密钥。
{{% /alert %}}

1. 在左侧导航栏，选择 **_"主机/主机/虚拟机"_** 菜单项，进入虚拟机页面。
2. 单击虚拟机右侧操作列 **_"更多"_** 按钮，选择 **_"密码密钥-解绑密钥"_** 菜单项，弹出绑定密钥对话框。
2. 默认勾选“绑定密钥后自动启动”，可根据需要取消勾选，单击 **_"确定"_** 按钮，完成操作。
