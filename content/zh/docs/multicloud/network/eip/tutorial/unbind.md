description: >
    解除弹性公网IP与资源的绑定
---

解除弹性公网IP与资源的绑定，解绑后的弹性公网IP可以绑定到其他资源。

{{% alert title="注意" color="warning" %}}
绑定AWS NAT网关的EIP暂不支持解绑
{{% /alert %}}

1. 在左侧导航栏，选择 **_"网络/网络服务/弹性公网IP"_** 菜单项，进入弹性公网IP页面。
2. 单击弹性公网IP右侧操作列 **_"更多"_** 按钮，选择下拉菜单 **_"解绑"_** 菜单项，弹出解绑确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。
