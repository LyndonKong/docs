title: "删除文件（夹）"
date: 2021-11-26T17:03:20+08:00
weight: 70
description: >
    删除存储桶下的文件或文件夹
---

支持单个或批量删除文件(夹)。

**单个删除**

1. 在左侧导航栏，选择 **_"存储/对象存储/存储桶"_** 菜单项，进入存储桶页面。
2. 单击指定存储桶名称项，进入文件列表页面。 
2. 单击文件（夹）右侧操作列的 **_"删除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在文件列表中勾选一个或多个文件（夹），单击 **_"删除"_** 按钮，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。
