description: >
    下载存储桶内的文件到本地
---

该功能支持用户下载文件到本地。

1. 在左侧导航栏，选择 **_"存储/对象存储/存储桶"_** 菜单项，进入存储桶页面。
2. 单击指定存储桶名称项，进入文件列表页面。 
2. 单击文件右侧操作列的 **_"下载"_** 按钮，浏览器下载或打开文件。
3. 若下载的文件格式浏览器支持打开，浏览器将会直接打开文件而不会下载，建议在浏览器上将文件另存为，并保存到本地。
4. 若下载的文件格式浏览器不支持打开，浏览器将会直接下载文件。
