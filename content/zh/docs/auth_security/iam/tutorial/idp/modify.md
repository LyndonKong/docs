    修改认证源配置

该功能用于修改认证源的配置信息，只有禁用状态的认证源支持修改配置。

{{% alert title="注意" color="warning" %}}
- SQL认证协议的认证源不支持修改配置。
{{% /alert %}}

1. 在认证源页面，单击"禁用"状态的认证源右侧操作列 **_"修改配置"_** 按钮，进入修改配置页面。
2. 修改不同认证类型的认证源的参数，修改完成后，单击 **_"确定"_** 按钮。
