该功能用于删除用户。

**单个删除**

1. 在左侧导航栏，选择 **_"认证与安全/认证体系/用户"_** 菜单项，进入用户页面。
2. 单击用户右侧操作列 **_"更多"_** 按钮，选择下拉菜单 **_"删除"_** 菜单项，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在用户列表中勾选一个或多个用户，单击列表上方 **_"批量操作"_** 按钮，选择下拉菜单 **_"删除"_** 菜单项，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。
