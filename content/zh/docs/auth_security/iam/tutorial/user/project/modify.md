该功能用于修改用户在项目中的角色。

{{% alert title="注意" color="warning" %}}
通过组加入的项目不支持修改角色。
{{% /alert %}}

1. 在用户页面，单击用户右侧操作列 **_"管理项目"_** 按钮，进入详情-管理项目页面。
2. 单击所属项目右侧操作列 **_"修改角色"_** 按钮，弹出修改角色对话框。
3. 修改角色后，单击 **_"确定"_** 按钮，完成操作。
