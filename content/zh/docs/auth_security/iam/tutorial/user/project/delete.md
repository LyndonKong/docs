该功能用于将用户移出项目。

{{% alert title="注意" color="warning" %}}
通过组加入的项目不支持移出项目。
{{% /alert %}}

**单个移除**

1. 在用户页面，单击用户右侧操作列 **_"管理项目"_** 按钮，进入详情-管理项目页面。
2. 单击所属项目右侧操作列 **_"移除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，将用户移出项目。

**批量移除**

1. 在用户页面，单击用户右侧操作列 **_"管理项目"_** 按钮，进入详情-管理项目页面。
2. 在列表中选择一个或多个所属项目，单击列表上方 **_"移除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，将用户移出项目。
