---
title: "删除云用户"
date: 2021-12-08T15:30:57+08:00
weight: 20
description: >
    删除公有云上的对应子账号
---

该功能用于删除云用户，删除操作将会在对应公有云平台上删除对应用户。

**单个删除**

1. 在左侧导航栏，选择 **_"认证与安全/认证体系/用户"_** 菜单项，进入用户页面。
2. 单击用户名称项，进入用户详情页面。
2. 单击“云用户”页签，进入云用户页面。
3. 单击云用户右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在左侧导航栏，选择 **_"认证与安全/认证体系/用户"_** 菜单项，进入用户页面。
2. 单击用户名称项，进入用户详情页面。
2. 单击“云用户”页签，进入云用户页面。
3. 在列表中选择一个或多个云用户，单击列表上方 **_"删除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，完成操作。
