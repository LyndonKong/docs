该功能用于将用户移出组。

**单个移除**

1. 在左侧导航栏，选择 **_"认证与安全/认证体系/组"_** 菜单项，进入组页面。
2. 单击组右侧操作列 **_"管理用户"_** 按钮，进入详情-组内用户页面。
2. 单击用户右侧操作列 **_"移除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，将用户移出组。

**批量移除**

1. 在左侧导航栏，选择 **_"认证与安全/认证体系/组"_** 菜单项，进入组页面。
2. 单击组右侧操作列 **_"管理用户"_** 按钮，进入详情-组内用户页面。
2. 在列表中选择一个或多个用户，单击列表上方 **_"移除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，将用户移出组。
