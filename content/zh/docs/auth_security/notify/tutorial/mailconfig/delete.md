该功能用于删除通知渠道。

**删除**

1. 在左侧导航栏，选择 **_"认证与安全/消息中心/通知渠道设置"_** 菜单项，进入通知渠道设置页面。
2. 单击通知渠道右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在通知渠道列表中选择一个或多个通知渠道，单击列表上方 **_"删除"_** 按钮，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。
